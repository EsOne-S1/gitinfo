Клонирование проекта в папку nameProject:
git clone https://... nameProject

Добавить файлы коммит:
git add.

Записать коммит:
git commit -m "Комментраий коммита"

Отправить коммит:
git push

Принять изменения:
git puii

Авторство коммитов:
git config --global user.email "Ваш емаил"
git config --global user.name "Ваше имя"
